# Prompts

All system-defined Prompts should be configurable! They must appear on the System Prompts page, and must allow the user to define a specific model for use with the prompt.

## Redirection

When the user is triggering a prompt, it is ideal to provide a place to use the generic 'Edit System Prompt' button, to link them to the prompt page! We should ideally allow them to return without losing page state.
